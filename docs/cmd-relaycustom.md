# /relaycustom

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/relaycustom <text>
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Adds &lt;text&gt; to the custom filter section of the INI file and will not relay any tells that include &lt;text&gt;.
<!--cmd-desc-end-->
